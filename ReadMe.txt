Simply type the following into the terminal/ cmd prompt to use the server:


server.py in Python

The server should keep running until it encounters an error. To illustrate progress and other pertinent information, various outputs are printed. Use C to exit the server software.



Simply run the following commands in a different terminal/ cmd prompt to utilise the client:



client.py in Python

A total of four commands are available to the client.

connect - this command establishes a connection between the client and the server.

upload - transmits the requested file from the client to the server.

get - this command downloads a file from the server and sends it to the client.

quit - this command disconnects the client from the server and terminates the client application.

To exit the client software, type quit.
